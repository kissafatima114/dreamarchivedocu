Tools, Effects, and Interface Design

Dream Archive was developed using HTML, CSS, and JavaScript as a browser-based interactive environment. Earlier versions of the project were created using p5.js to experiment with distortion techniques and image transitions before moving into a multi-page website structure.

The project uses layered visual effects including blur transitions, distortion filters, animated overlays, and atmospheric background elements to create the feeling of unstable perception. A scroll-based eyelid masking effect simulates the opening and closing of vision, reinforcing the idea of drifting in and out of memory. Additional interface elements such as floating particles, cursor glow interaction, and noise textures contribute to the overall atmosphere of the archive space.

Rather than functioning as a traditional navigation system, the interface encourages slow movement between entries so that visitors experience the archive as an environment rather than a sequence of pages.
